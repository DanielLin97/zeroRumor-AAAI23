Code and datasets for our AAAI 2023 paper - Zero-Shot Rumor Detection with Propagation Structure via Prompt Learning will be released soon...

## Requirements
```
python==3.8.10
pandas==1.1.5
numpy==1.23.5
protobuf==3.19.4
torch==1.8.0
```
